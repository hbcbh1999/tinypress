Welcome to GitHub Pages.

What's this?

TinyPress allows you create and manage your blog on Github via Github pages. Github pages are simple static websites [and/or blogs] for you and your projects, freely hosted and published through Github.

TinyPress features a clean, clutter-free interface to create a page (if you don't have one already), edit, delete and create new posts.

What information about me is accessed via my Github account?

TinyPress requests access to your Github public repos to authenticate you, check if you have a Github page blog and pull your posts. If however you don't have a Github page, TinyPress will request access to your personal repo to be able to create your Github page.

What information is stored?

To make TinyPress work as is, your public Github profile details (username, name, email) and access to your public repo are stored. Posts are also cached (temporarily stored) during the active session. For more information about this, check out the privacy policy.
